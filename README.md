# ðŸ’» Running the project

to run this project 
Clone this repository :
```
git clone [https://github.com/abderrahmaneGasmi/PaintSocket]
```
## Backend
```
cd paintsocketback
```
Install packages :
```
npm i
```
When installation is complete, run command :
```
npm start
```
## Frontend
```
cd paintsocketfront
```
Install packages :
```
npm i
```
When installation is complete, run command :
```
npm run dev
