to run this project 
Clone this repository :

## Backend
```

```
Install packages :
```
npm i
```
When installation is complete, run command :
```
npm start
```
## Frontend
```
cd paintsocketfront
```
Install packages :
```
npm i
```
When installation is complete, run command :
```
npm run dev
